# 2-way-dictionary
Алтернатива на нерастящия, но и остаряващ KBGOFFICE двупосочен речник

# Описание
То не знам какво толкова има да пиша тук. Вероятно всички знаем за англо-българския двупосочен речник KGBOFFICE. Той обаче е връстник на Лили Иванова и поради това библиотеки и модули периодично му изчезват и се налага да ги търсиш, за да ги връщаш обратно. Тази игра на котка и мишка няма да трае вечно и затова с малко помощ от AI (този път много по-малко, отколкото в други проекти) направих тази двупосочна алтернатива.

# Зависимости
Хубавото на тази алтернатива е, че няма нужда от компилиране или инсталация, само трябва да имате инсталиран GTK3, QT5 или QT6 + python-pyqt5 & python-pyqt6.
Алтернативата съм я кръстил "[insert framework here] 2-Way Dictionary".

# Сваляне и подкарване
Архивите за сваляне са горе - над readme файла.
Програмата може да се подкара по два начина:

• $ python3 en-bg-dict.py

• $ sudo chmod +x en-bg-dict.sh && ./en-bg-dict.sh

Ако ви кефи, можете да си направите и десктоп файл, където за Exec= посочвате (без кавичките) "python3 en-bg-dict.py".
